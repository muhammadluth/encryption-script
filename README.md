# encryption-script

This is a app for encryption data, you can integration with postman, jmeter, etc.

## Features

> - [x] RSA SHA-256 Hex
> - [x] RSA SHA-256 Base64
> - [ ] RSA SHA-512 Hex
> - [ ] RSA SHA-512 Base64
> - [ ] HMAC SHA-256 Hex
> - [ ] HMAC SHA-256 Base64
> - [ ] HMAC SHA-512 Hex
> - [ ] HMAC SHA-512 Base64
